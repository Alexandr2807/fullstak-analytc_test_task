# BA/SA Test Task

Выполненное тестовое задание на позицию БА/СА аналитика.

## Состав решения

- Task 1 — BPMN 2.0 диаграмма процесса выдачи ИТ-оборудования
- Task 2 — User Story и Use Case для публикации товара на маркетплейсе
- Task 3.1 — описание REST API для регистрации пользователя
- Task 3.2 — пошаговый алгоритм создания пользователя на стороне backend

## Файлы репозитория

- `task1_bpmn.png` — BPMN диаграмма
- `task2.md` — требования к новой функциональности публикации товара
- `task3_api.yaml` — OpenAPI спецификация метода регистрации
- `task3_algorithm.md` — алгоритм обработки регистрации пользователя

## Примечание

В BPMN-модели отдельно отмечены вопросы и возможные пробелы в исходном описании процесса, которые важно уточнить для корректного отражения процесса "as is".
